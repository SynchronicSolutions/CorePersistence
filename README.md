# Requirements

- iOS 10.0+
- Xcode 8.3+
- Swift 5+

# Installation

CorePersistence is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CorePersistence'
```
